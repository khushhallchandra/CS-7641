The code can be found at https://github.com/khushhallchandra/CS-7641.

Clone the github repository using 'git clone https://github.com/khushhallchandra/CS-7641.git'

The data folder contains the wine dataset. The MNIST dataset is imported from keras library.

All the required packages are given in the requirements.txt file.

Please install it using `pip install -r requirements.txt`

The folder conatains files named with followig convention: algo_dataname.ipynb

Then run the notebook as:

jupyter notebook filename

For ex:

jupyter notebook DecisionTrees_mnist.ipynb
